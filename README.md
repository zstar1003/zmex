# 择木而栖 ZMEX

面向高考志愿筛选的静态可视化网站，提供全国院校分布地图、专业录取位次参考和专业倾向测评。站点优先使用可追溯的公开数据，无法取得完整官方专业线时会明确标注数据边界。

线上站点：https://zmex.zstar.website/

## 功能

- 全国院校地图：覆盖教育部《全国普通高等学校名单》中的普通本科院校和高职（专科）院校。
- 院校筛选：支持按省份、办学层次、本科/专科、办学性质、重点标签和院校类型筛选。
- 排名估算：按“省份 + 科类 + 全省排名”查询可参考专业，并支持限定院校所在省份。
- 专业详情：点击院校或专业后展示专业排名参考、已整理的专业录取位次和原始来源入口。
- 专业倾向测评：提供轻量的兴趣倾向问卷，辅助缩小专业方向。

## 运行

本项目使用 Bun。

```bash
bun install
bun run start
```

打开 `http://localhost:5173/`。

## 数据

主院校数据来自教育部《全国普通高等学校名单》（名单截至 2025-06-20，公告发布于 2025-06-27）。当前生成结果：

- 普通高校总数：2919 所
- 本科院校：1365 所
- 高职（专科）院校：1554 所
- 覆盖省级地区：31 个
- 覆盖城市：347 个

核心数据文件：

- `data/schools.json`：前端主院校数据，含本科和专科。
- `data/china.json`：中国地图 GeoJSON。
- `data/raw/moe-colleges-2025.xls`：教育部原始附件。
- `data/raw/moe-colleges-2025.rows.json`：由 Excel 转出的二维数组。
- `data/admissions/sources.json`：各省投档数据源登记表。
- `data/admissions/provinces/`：已取得官方专业线省份的规范化数据。
- `data/admissions/aggregated/provinces/`：公开专业录取数据与来源索引。
- `scripts/build-data.mjs`：生成院校地图数据。
- `scripts/sync-admissions.mjs`：下载、校验并规范化官方投档数据。
- `scripts/sync-gaokao-cn.mjs`：逐省逐校查询公开专业录取数据。
- `scripts/verify-admissions.mjs`：校验全国记录数量、来源、重复 ID 和位次字段。

重新生成院校数据：

```bash
bun run build:data
```

同步已登记的官方投档表：

```bash
bun run sync:admissions
```

查询公开专业录取数据：

```bash
bun run sync:aggregated
```

校验全国招生数据：

```bash
bun run verify:admissions
```

## 位次数据规则

排名查询以“省份 + 院校 + 专业”为数据粒度，不用一所学校的最低门槛代替全部专业。数据按以下顺序使用：

1. 浙江、山东、河北、辽宁、重庆使用 2025 年省级考试机构官方专业投档数据。
2. 其余多数省级招生地使用 2024 年掌上高考公开专业录取数据，保留每所学校静态 JSON 地址和 SHA-256 文件哈希。
3. 新疆多数专业只公开最低分，使用公开转载的 2024 年文理科一分一段表换算同分位次上限，并校验每一分档的累计递推关系。
4. 西藏没有公开一分一段表，公开专业数据连续多年也不提供位次。470 条西藏普通本科记录仅把公开最低分换算为粗略估算位次，页面单独标注“估算位次”。
5. 湖北已找到 2025 年本科普通批院校/院校专业组投档线，但不是具体专业录取位次表，因此当前专业推荐仍使用可追溯的 2024 年专业粒度公开记录。

普通本科专业线同步脚本仍只扫描本科院校；首页院校地图已经补入专科院校名单。
